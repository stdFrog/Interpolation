커스텀 에디트 제작으로 인해 스케줄러 프로그램 완성이 늦어지면서
스케줄 이미지라도 띄워놓으려고 만든 간단한 프로그램입니다.

작업 영역에서 우클릭시 비트맵 파일을 읽어올 수 있습니다.
읽어온 비트맵은 화면에 출력됩니다.

사이즈 조정시 비트맵 특성에 의해 색상 비트가 겹치거나 이미지가 뭉개지는 경우가 있는데 이를 보정하는 함수도 추가했습니다.
사실 직접 만들 필요없이 API가 지원하는 StretchBltMode 함수를 사용하면 훨씬 선명한 이미지를 보여줍니다.

다만, 꽤 어렵게 공부하기도 했고 복습할 겸 직접 유틸리티 함수를 만들었습니다.

소스 코드를 분석하려면 비트맵 구조와 색상에 대한 기본적인 지식이 필요합니다.
보간법 함수도 분석해보면 꽤 도움이되니 참고하시기 바랍니다.
